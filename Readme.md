# Getting Started
1. download python from python.org
2. check that python is working on your machine with `python3 --version` or `python --version` 
3. check that pip is installed on your machine with `pip3 --version` or `pip --version`
4. run `python3 -m pip install pygame --user`
5. check that pygame has been installed with `python3 -m pygame.examples.aliens`
